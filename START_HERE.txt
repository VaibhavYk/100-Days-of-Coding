FEATURES EXPLAINED:
===================

1. Public IP Address
   ------------------
   Shows your IP address as seen by the internet. This is assigned by your
   Internet Service Provider (ISP). Useful for:
   - Setting up remote access
   - Troubleshooting network issues
   - Checking if you're behind a VPN

2. Local IP Address
   -----------------
   Shows your IP address on your local network (home/office). Usually starts
   with 192.168.x.x or 10.x.x.x. Useful for:
   - Setting up local servers
   - Network troubleshooting
   - Connecting devices on same network

3. Ping Website
   -------------
   Sends test packets to a website and measures response time. Shows:
   - If the website is reachable
   - How fast the connection is (latency)
   - Packet loss (if any)
   
   Lower ping = better (15-30ms is excellent, 50-100ms is good)

4. Internet Speed Test
   --------------------
   Downloads a test file and calculates your download speed in Mbps.
   Note: This is a basic test. For accurate results, use speedtest.net

TROUBLESHOOTING:
================

Problem: "gcc is not recognized"
Solution: Install MinGW-w64 and add to PATH
          See WINDOWS_SETUP.md for detailed instructions

Problem: "curl not found"
Solution: Install curl (included in Windows 10+)
          Or download from https://curl.se/windows/

Problem: "Failed to initialize Winsock"
Solution: Run as Administrator
          Right-click â†’ Run as administrator

Problem: Compilation errors
Solution: Make sure you include both libraries:
          -lws2_32 -liphlpapi

Problem: No local IP displayed
Solution: Check network connection is active
          Try running as Administrator

Problem: Ping fails
Solution: Check firewall settings
          Enable ICMP in Windows Firewall
